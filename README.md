# TaleBox 📖✨

> **你的掌上 AI 互动小说世界**

TaleBox 是一款专为手机端设计的 **AI 互动小说阅读与创作应用**。在这里，你不仅是置身事外的读者，更是剧情的决策者，是小说角色的对话者，更是整个故事世界的创造者。

---

## 🌟 核心产品体验

### 💬 沉浸式角色互动对白
告别单向的文字阅读。在 TaleBox 中，你可以像和朋友聊天一样，直接与小说中的角色进行实时交谈。角色会严格遵循自身的人设，给予你最真实、最具情感的个性化回复。

### 🎭 动态剧情，由你掌控
拒绝一成不变的套路剧情。你的每一次选择、每一句回答，都将实时影响故事的发展方向。AI 将根据你的反馈，智能规划并稳定推进剧情，为你量身定制独一无二的阅读体验。

### 🌍 零门槛私密小说创作
每个人都是创作者。你只需输入一个灵感片段，TaleBox 就能帮助你构建出宏大的世界设定、多维的角色卡片，并自动生成引人入胜的第一幕场景，开启属于你的原创小说旅程。

### 📖 让你的本地小说“活”过来 *(未来规划)*
将你喜爱的 TXT 或 EPUB 格式的本地小说导入应用，TaleBox 将利用 AI 技术自动解析并生成可交互、分叉的剧情，让你能亲身参与到原作人物的命运抉择中。

---

## 🔄 核心产品闭环

TaleBox 的设计理念围绕着一个极简而流畅的体验闭环：

```
💡 创意闪现 ➔ 👥 诞生角色 ➔ 💬 开启对话 ➔ 🔮 剧情推进 ➔ 💾 保存状态 ➔ 🔄 随时继续
```

- **私密空间**：我们坚信阅读与创作是个人的、自由的。在第一阶段，TaleBox 不提供社交媒体流、好友评论区、公开排行榜等功能。您的故事世界完全属于您自己。
- **状态可溯**：每一次对话都会被记录并演化为故事状态，您可以随时保存当前进度，下次继续沉浸在未完的章节中。
