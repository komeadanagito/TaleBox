# TaleBox 📖✨

![TaleBox Banner](image.png)

> **你的 AI 互动小说世界**

TaleBox 是一款 **AI 驱动的互动小说平台**。在这里，你不只是一个旁观的读者——你是剧情的决策者，是角色的倾听者，更是整个故事世界的创造者。

---

## ✨ 核心体验

### 💬 与角色实时对话
抛开单向阅读的隔阂。在 TaleBox 中，你可以直接与小说中的角色聊天——他们有自己的记忆、情绪与人设，每一次回应都真实而富有温度。

### 🎭 你的选择，决定故事走向
每一个决定都会真实地影响剧情。AI 会记住你说过的每一句话、做过的每一个选择，并为你量身推进一段独一无二的故事。

### 🌍 零门槛，开始创作
一个灵感片段就够了。TaleBox 帮你构建世界观、生成角色卡、写出第一幕——创作从未如此轻盈。

### 📖 让本地小说"活"过来 *(即将上线)*
导入你喜爱的 TXT / EPUB 小说，TaleBox 自动解析剧情，让你参与到原著人物的命运抉择中。

---

## 🔄 体验闭环

```
💡 灵感涌现  →  👥 角色诞生  →  💬 展开对话  →  🔮 剧情推进  →  💾 存档继续
```

- **私密空间**：没有公开动态流，没有评论区压力。你的故事，只属于你。
- **随时继续**：每段对话都会演化为故事状态，随时保存、随时重拾。

## 🚀 快速部署

> 三步启动，开箱即用。

**1. 克隆项目**

```bash
git clone https://github.com/your-org/talebox.git
cd talebox
```

**2. 配置环境变量**

复制根目录的 `.env` 模板，填入你的 API Key：

```bash
cp .env.example .env
# 编辑 .env，填写 LLM / 数据库等所需配置
```

**3. 启动**

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:3000` 即可体验。
